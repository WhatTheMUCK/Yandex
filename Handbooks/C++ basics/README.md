# Повторение основ C++ по хэндбуку от Яндекса

В рамках изучения [хэндбука по C++ от Яндекса](https://education.yandex.ru/handbook/cpp) мной были пройдены следующие темы:

- **Базовые конструкции C++**: первые шаги, типы данных, ветвления и циклы, работа с векторами и строками, функции.
- **Стандартная библиотека C++**: последовательные и ассоциативные контейнеры, алгоритмы, адаптеры и представления.
- **Идиомы C++**: создание и использование классов, шаблонные классы, жизненный цикл объектов, наследование и полиморфизм, обработка исключений, RAII и умные указатели.

Все теоретические разделы прочитаны, а практические задачи выполнены. Ниже перечислены решённые задачи с ссылками на соответствующие каталоги.
